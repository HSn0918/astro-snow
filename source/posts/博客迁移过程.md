今天是 2025-12-27 

我的博客系统终于升级啦！仓库见 https://github.com/XueHua-s/astro-snow 。

对余弦小姐的博客进行了一部分改动：
- 删掉一些大多数人不需要的功能
- 增加并调整了一些内容和配置

具体改动参考 https://github.com/XueHua-s/astro-snow 仓库。

博客自从 22 年搭建以来，历经了多次变化，从 asp 到 next 到 极简主义 Hexo。

最后就是现在非常二次元、花里胡哨的二次元风博客啦！

README.md 里提到的改动（精简版）：
- 增加引导页与 GitHub Actions 自动部署 / AI 摘要
- 配置集中到 `src/config`，新增 @ 路径别名
- 移除 RSS/XML、周刊、Mermaid 优化脚本、二级分类
- 文章页按 `date` 自动生成，分类按 `categories` 构建
- 重逻辑组件改为 React，优化 Hook 与部分结构
- 评论改为 giscus，新增随机背景图

更换使用 Astro，相比之前 Hexo 的好处：
- 不同于 Next 的 SSG，它加载更快；纯内容、首屏无 JS，不是完全水合。
- 复杂逻辑可以使用 React / Vue，以及相应生态里的工具和组件。
- astro-pagefind, 系统自带无后端搜索(通过生成的索引文件进行搜索)


总之就是就是Astro太香了. cosine姐的模板, 也很有特色。

比如: 

- 支持ai摘要
- 图片LQIP（低质量图片占位符），图片加载前显示渐变色占位

也很好看.最适合我这种肥宅了.

然后，我还配置了GithubAction, 上传到GithubPage托管(其实之前博客就有), 同时每次在打包时进行ai摘要。
